Shelf Scan: Automated Book Spine Recognition and Cataloging

This system uses OpenCV for detecting and cataloging book spines. It features vertical line detection and OCR techniques to extract book titles accurately. Please note that this is a raw image processing algorithm, not a machine learning method. As a result, the accuracy may vary, and sometimes the results may not be perfect. For better results, ensure that only one rack of books is included in the image and crop the bookshelf accordingly.

Project Details

Developed by: Sakthi Avinash
Student at Vellore Institute of Technology, Chennai

In this website, I share my journey of developing a test model that uses a peak-driven method to detect book spines. This method focuses on identifying the peaks of book spines, which is important for accurate text extraction. Through my experiments, I explore how this model works and how it compares with traditional methods like Canny contour edge detection. This new approach has the potential to improve book spine detection in certain situations.
